#🚀 GSAP ScrollTrigger Smooth Scrolling Website

This is my first GSAP ScrollTrigger-based scrolling website, built to explore modern animations and smooth scrolling techniques. The project focuses on creating visually engaging scroll-based animations using industry-standard libraries.

---

#🛠 Tech Stack

HTML5 – Structure of the website

Tailwind CSS – Utility-first CSS framework for fast and responsive UI

JavaScript (Vanilla JS) – Core logic and interactions

GSAP (GreenSock Animation Platform) – High-performance animations

GSAP ScrollTrigger – Scroll-based animation control

Framer Motion – Smooth UI animations and transitions

Lenis – Smooth scrolling experience

---

#✨ Features

Smooth scrolling using Lenis

Scroll-based animations powered by GSAP ScrollTrigger

Clean and responsive UI with Tailwind CSS

Modern motion effects using Framer Motion

Optimized animation performance

Beginner-friendly GSAP implementation

#⚙️ How It Works

Lenis handles smooth scrolling behavior

GSAP ScrollTrigger syncs animations with scroll position

Framer Motion enhances UI transitions and element animations

Tailwind CSS ensures fast styling and responsive layouts
